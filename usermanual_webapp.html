Overview

The p-Chip Cloud web application is a cloud hosted platform where clients can set-up and manage their organization and access the historical information of all of their products using p-Chip microtransponder technology across the supply chain.

Getting Started

This guide will help you as your organizations administrator get up and running quickly to start experiencing the incredible benefits that securing your products with p-Chip microtransponder technology offers to you and your customers

To read more about the benefits of p-Chip security start your journey here.

Pre-Requisits

  • Google Chrome, Microsoft Edge or other compatible web browser
  • Microsoft Office Account

Requesting an Account

  1. In your web browser, go to http://pchip-web-portal.azurewebsites.net/
  2. Select Request Account
  3. Fill in the form with your companies information
  4. Submit the form by selecting the Request Account button when complete

Logging In

  1. In your web browser, go to http://pchip-web-portal.azurewebsites.net/
  2. Select Sign in With Office 365
  3. Allow the p-Chip Cloud to read your Microsoft Profile by selecting Yes when prompted

Settings

The application settings is where you can control your organizations information. This information is required for security purposes. This information can be changed by your company's administrator at anytime.

Components

  • Organization Name: The registered name of your business or organization
  • Email: Company email
  • Telephone: Company telephone number
  • Street: Street: Registered street address
  • City: City: Registered city
  • Post Code: Registered postal code
  • Country: Registered country
  • Service Agreements: Terms of service for use of the application

User Interface Guide

Homepage

Once logged in you will be presented with your organizations homepage. This is the core of the application from which you can manage your organizations supply chain tracking with p-Chip Cloud. You can navigate through the application using the tabs on the left hand side of the homepage.

Components

  1. Dashboard: The dashboard shows you an overview of your organizations products tags with p-Chip. You can view the last scanned location and view the historical global activity of your products tagged with p-Chip microtransponder tags
  2. Users: Web application users
  3. Licenses: p-Chip Cloud Desktop Application licenses issued by your organization
  4. Groups: License groups
  5. Products: Products that can be associated with p-Chip microtransponder tags
  6. Tests: Define tests for products and record the results in the cloud
  7. Tags: Your organizations products using p-Chip microtransponder tags
  8. Tag Activity: Historical information of all scans performed by organization licenses
  9. Error Reports: Issues and error reporting from p-Chip Cloud Desktop Application users.
  10. Organizations (Super Admin Only): p-Chip client organizations
  11. Account Requests: (Super Admin Only): Requests for Organization account access made via the web portal
  12. Tracked Items: Total number of p-Chip microtransponder tags used by your organization
  13. Active Items: Number of p-Chip microtransponder tags with an Active status
  14. In-Active Items: Number of p-Chip microtransponder tags with an In-Active status
  15. Activity: Total number of Activities or scans by p-Chip Cloud desktop application users
  16. Tracked Items: The last scanned location for your organizations items tagged with p-Chip microtransponder technology visualized by location
  17. Tracking Activity: Time series chart showing the overall Activities performed by p-Chip Cloud Desktop App users across your organization

The p-Chip Cloud web application is designed to give an organization full control and visibility of their supply chain. To achieve this the web portal allows administrators to perform a series of functions that enable p-Chip Cloud Desktop App users to perform tasks such as incoming and outgoing invventory checks, quality control and much more. Below are a list of functions currently available to Organization Administrators.

Dashboard

The p-Chip Cloud Dashboard is the command center for managing your global supply chain. It is designed to give you a high level understanding of where your products are, the overall activity of your supply chain and the number or percentage of goods lost or Inactive.

Components

  1. Tracked Items: Total number of p-Chips applied and recorded by your organization in the p-Chip Cloud tracking system.
  2. Active Items: Total number of p-Chips with a status of Active
  3. Inactive Items: Total number of p-Chips with a status of Inactive
  4. Activity: Total number of scans of p-Chips performed by users with active licenses across your organization.
  5. Tracked Items (Map): Last scanned location of p-Chip microtransponder tags created by your organization.
  6. Tracking Activity (Bar Chart): Total number of scans of p-Chips performed by users with active licenses across your organizations plotted over time.

Users

The users section allows you to add web application users to your organization. These users have full access and permissions to manage the organizations account. The Users section can be accessed via the left naviation pane.

Components

  1. Users tab: Accesible main navigation
  2. Filter pane: Filter the list of organization users by their status, role or organization
  3. Add User: Add users to your organization
  4. User List: List of all users with access to your organization

Adding Users

  1. From the User tab, select + Add User
  2. Enter the users First Name
  3. Enter the users Last Name
  4. Enter the users Email Addres
  5. The Organization and Profile are populated by default
  6. Select Create to save the user to your organization
  7. Select the User you have just created from the User List
  8. Copy and paste the Activation Code from the User Details view
  9. Instruct the user to log-in and enter the Activation Code when prompted
  10. p-Chip Admin Only
    • Select the users Organization if you are creatinga user on behalf of a Client Adminstrator

View User Details

  1. Select a User from the User List
  2. Available User Information:
    • Name
    • Email
    • Activation Code
    • Organization
    • Status
    • Role
  3. The following lists show the entries created by the user:
    • Users
    • Licenses
    • Products
    • Groups
    • Tests
  4. Select Back to return to the User List

Editing User Information

  1. Select a User from the User List
  2. Select Edit
  3. You can edit the following user information:
    • First Name
    • Last Name
    • Email

Deactivate Users

For security reasons, users cannot be removed from the system. However, their access rights can be removed at anytime by deactivating their account. Once deactivated, the user will no longer have access to the p-Chip Cloud Web App or your Organizations information.

  1. Select a User from the User List
  2. Select Deactivate

Licenses

Licenses are used by the p-Chip Cloud Desktop Application to authorize your organizations devices to create and search for your products with p-Chip microtransponder tags. Licenses are issued on a device basis meaning that they cannot be reused across devices. Licenses are issued by the client administrator through the web portal. Licenses that have been issued to your organization and are no longer in use can be deacativated at anytime.

Components

  1. Your organization List of Licenses are accessible via the main navigation menu
  2. Filter your organization list of licenses by
    • Organization (p-Chip Admin Only)
    • Created by User
    • License Number
    • License Status
  3. Add Licenses to your organization at anytime
  4. View a full list of licenses issued by your organization

Issuing Licenses

  1. Select the + Add License button
  2. Copy and Paste the License Number and License Key and send to the user of the device that you would like to activate
  3. Enter a meaningful description to whom you have provided the license. This will help you in the future if you need to deactivate a specific user or device access rights
  4. Select Create

View License Details

  1. Select a License from the License List
  2. Available License Information:
    • License Number
    • License Key
    • Created Date
    • Created By (User)
    • Assigned To
  3. The following lists show information about the license access and activity:
    • Groups: Groups to which the license belongs
    • Tags: Tags created by the license
    • Tag Activity: All scans and activities performed by the license
  4. Select Back to return to the License List

Assign Licenses to Groups

Licenses and groups are how we ensure the security of data within an organization. If you are an organization with 300 locations, your users don't need nor want to see the activity of all 300 locations. To ensure the user only sees what they need to see, assigning licenses to groups cuts out the noise by only showing the user what they need to see. One license can belong to 1 or multiple groups.

  1. Select a License from the License List
  2. Select + Assign Group
  3. Select 1 or more Groups from the available list by selecting the checkbox
  4. Select Assign Selected Groups to License

Deactivating Licenses

  1. Select a License from the License List
  2. Select Deactivate in the upper right hand corner
  3. Select Back to return to the License List

Groups

Groups are how data is segregated with an organization. Groups can be a phystical location, a department or any other type of a group of devices. Groups allow p-Chip Cloud Desktop App users to share data, collaborate and stay up to date with each others activities no matter where they are in the world. Groups reside within an organization and cannot span outside of the organization.

Components

  1. Your organization List of Groups are accessible via the main navigation menu
  2. Filter your organization list of Groups by

      • Country

      • Street
      • City
      • Postal Code
      • Country
    • Department
  3. Select Create

View Group Details

Products

Products are physical goods or items that your organization wants to track across the supply chain or for the service life of the item such as equipment or devices. Products are created within an organization and assigned to various groups. When a product is assigned too a group, group members can create or convert items with p-Chip microtransponder tags and store this information in the p-Chip Cloud so it is accessible across your organization.

Components

  1. Your organization List of Products are accessible via the main navigation menu

Common examples include: "Not Started", "In Process", "Quality Control", "Approved", "Rejected". These statuses can be customized to your own organization and to the product itself for greatest possible flexibility.

Deactivating a Tag

The p-Chip status, a value of Active or Inactive, is used to control the tracking status of the item. If an item is lost, stolen or is not fit for use or sale, the tag can be deactivated which notifies the manufacturing and supply chain that the attached item is no longer authenticated by your organization. To deactivate a tag:

  1. Select a Tag from the Tag List
  2. Select Deactivate in the upper right hand corner
  3. Select Back to return to the Tag List

Tag Activities

Tag activities are specific events that are recorded in the p-Chip Cloud and made accessible across your organization. There are currently 4 activities currently supported. These include:

  • Add: A tag is added to your organization
  • Product Add: A product is added to an existing tag in your organization. A single p-Chip can be "converted" into multiple products during it's life cycle, just as goods may change from raw materials to intermediates to finished goods. Each transformation of a good is captured with the Product Add event.
  • Test: A test is performed on a tag.
  • Search: A tag is searched by a user within your organization.

During each of these activities, metadata on time, location, device, equipment and user information is collected and stored improve visibility across your global supply chain.

Components

  1. Your organization List of Tag Activities are accessible via the main navigation menu
  2. Find specific items quickly by filtering on:
    • p-Chip serial number
    • External ID
    • Product
    • License
    • Group
    • Type
  3. Export a full list of your organizations Tag Activities in csv format
  4. View a full list of Tag Activities created by your organization

Error Reports

Error reports are user generated reports across your organization. These reports are created in the p-Chp Cloud Desktop App and are designed to allow quick and easy communication between your organizations administrators and field users. Here users can report equipment issues, product issues and more.

Organizations (p-Chip Admin Only)

Organizations are companies or businesses that are using the p-Chip Cloud platform for securing their supply chain. All organizations data is segmented so that their data is secure and can only be accessed by administrator permissions detailed in the Users and License sections.

Create an Organizations

When an client has been approved for an account, the first step is to create a new organization. This organization is where all future activities will be perforemd by the organizations administrators. To create a new organization, follow the steps below:

  1. Navigate to the Organization tab
  2. Select +Add Organization in the upper right hand corner
  3. Complete the Create Organization form
  4. Select Create

Remove Organization Access

  1. Select an Organization from the list of organizations
  2. Select Deactivate

Account Requests (p-Chip Admin Only)

Prospective customers may request account access using the Request Account function. When new requests are received, these will be reviewed internally. If approved, you can approve the account request by following the below steps.

Approve Account Requests

  1. Navigate to the Account Requests tab
  2. Select the relevant account request from the list
  3. Select Approve to approve the request
