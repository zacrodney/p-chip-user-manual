Introduction

The p-Chip Cloud is an integrated development platform for building state of the art applications on cloud infrastructure hosted by p-Chip Corporation. Using this platform p-Chip clients can use native applications from the p-Chip library or build their own solutions that can integrate with the most popular 3rd party applcations used by enterprise organizations today.

Files

The following files are Windows Installation files required to use the p-Chip Cloud Platform.

  • p-Chip Cloud - Windows Application v.2.2.2 (.msi)
  • p-Chip Cloud - USB Drivers (.msi)
  • p-Chip Reader - 4.1.0-beta (.msi)
