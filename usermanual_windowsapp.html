Overview

The p-Chip Cloud Desktop application is a Windows based desktop application that is integrated with the p-Chip Wand or Scanner. Users can create, edit and search for products using p-Chip microtransponder technology. Work performed in this application is uploaded to the p-Chip Cloud where it can be accessed by authorized users across your organization and with participating supply chain partners.

Getting Started

This guide will help you as p-Chip Cloud user get up and running quickly to start experiencing the incredible benefits that securing your products with p-Chip microtransponder technology offers to you and your customers.

To read more about the benefits of p-Chip security start your journey here.

Pre-Requisits

  • Device
    • OS: Windows 10 version 21H2
    • Processor: 1.5GHZ
    • Storage: 60 GB
  • Device Settings
    • Location Services Enabled
    • Region number format: English (USA)
  • p-Chip Wand
  • p-Chip Wand USB Drivers
  • USB-A Device Port
  • p-Chip Cloud Organization
  • p-Chip Cloud Group
  • p-Chip Cloud Products
  • p-Chip Cloud License

Installation

  1. Download the p-Chip Cloud Windows Application
  2. Complete the installation wizard

Set Up

  1. Enable location services for the p-Chip Cloud Windows Application. Visit here for instructions on how to enable location services
  2. Connect to a secure wifi network
  3. Open the p-Chip Cloud Windows application
  4. Open the Settings tab
  5. Select View next to License #

User Interface Guide

Navigation

The p-Chip Cloud Windows Application is designed to be easy to use across desktop and tablet devices. To navigate through the features and functions of the application, use the navigation pane on the left hand side of the screen.

Components

  1. Records List
  2. Add Records
  3. Add Product
  4. Test Records
  5. Search Records
  6. User Manual
  7. Error Report
  8. Settings

Records List

The Records List is the first window you see when the application opens. This view shows you all products tagged with p-Chip microtransponder tags in your group. Read more about groups.

Components

  1. Product List: List of products to which your license has access.
  2. Filters: Filter your list of products by selecting your filter criteria and pressing Apply. Filter based on the following criteria:
    • Groups
    • Product Name
    • Product Status
    • Tag Status
    • Internal ID
    • Created Date Range (From-To)
  3. Device Identifier: 4 digit code of your p-Chip Wand
  4. Sync Now: Manually sync your local data with your organization cloud data repository.
  5. Network Status: Your devices network status

Add Record Window

The Add records window is where you can apply a p-Chip to a product and create in in the local application for the first time.

Components

  1. Select Group: Select the group to which you want to assign this item.
  2. Select Product: Select the product to which you want to assign this item.
  3. Scanning Settings: Select from the available options:
    1. Single: Turn on the wand for a single scan. After you scan a single p-Chip, the wand will be turned off.
    2. Multiple: Turn on the wand and scan multiple records. The device will automatically shut off after the amount of time you indicate in the Settings option Auto Scanner Shutdown (seconds)
    3. Always On: Turn on the wand and scan multiple records. To turn off the device, press the Stop button.
  4. Start/Stop Button: Start (power on) and Stop (power off) the wand for scanning.
  5. Session List: List of p-Chip microtransponder tags scanned during one session (i.e. at one time when the window is open). After closing the window or navigating away from the current view, the list will be removed and visible in the records list.
  6. Row Count: Counts the number of rows the in the Session List.

Record Details Window

The Record Details window displays all recorded information about a tag and related product, as well as the Production, Test and Search activities performed by all users within your organization.

Components

  1. Tag Details: Summary of the p-Chip tag inlcuding the following information:
    1. p-Chip ID: 9 digit numeric code embeded in p-Chip microtransponder
    2. Tag Status: Microtransponder tag operational status maintained by client admninistrator
    3. External ID: Serialized inventory item or batch identifier
    4. Age: # of days since microtransponder tag was added or created in the system
    5. Tests: # of tests performed on the related product
    6. Product Code: Product Identification Code defined by client administrator
    7. Product Name: Product name defined by client administrator
    8. Product Description: General description of the product. Defined by client administrator
    9. Product Status: Client defined goods status defined by client administrator
    10. Product Age: # of days since the current product was assigned to the microtransponder tag
    11. Production Date (UTC): Date and time that the current product was assigned to the microtransponder tag
    12. Production Location: Geographical data collected when the current product was assigned to the microtransponder tag
    13. Last Scan Date (UTC): The last date that the tag was modified, tested or searched by the current group
    14. Last Scan Location: Geographical data collected when the tag was last scanned by a member of the current group
    15. Current Group: The group that is assigned as the current owner of the tag.
  2. Edit: see Edit Record Form
  3. Done: Close record details and return to list
  4. Related Tag Activity: Lists of activities performed by the current group members including: Production (Add Products), Tests and Search

Functions

The p-Chip Cloud Windows Application allows users to create and share information about products and equipment across local groups and global organizations. These functions are accessible to all licensed p-Chip Cloud users with the appropriate hardware supplied by p-Chip Corporation detailed in the Pre-Requisits section of this user guide.

Instructions are provided for how to perform each of the activities currently available.

List of Functions

  • Create Records
  • Edit Records
  • Add Products
  • Add Tests
  • Search Records
  • Report an issue

Create Records

Records are individual items to which a p-Chip has been applied. When you Create a Record with p-Chip, the p-Chip microtransponder serial number, metadata and your organizations user input is recorded into the p-Chip Cloud data platform.

Instructions

  1. Connect your p-Chip wand to your device
  2. Select the Add Record button from the navigation menu
  3. If you belong to more then 1 group select the Group that should be the owner of this item, otherwise your default group will automatically be entered and not editable.
  4. Select the Product from the dropdown menu.
  5. Select the Scanning Mode, multi-scan(recommended).
  6. Press start
  7. Press the tip of the wand against the p-Chip
  8. Stop scanner when all p-Chips have been scanned

Important!

  • A p-Chip can only be added to the system one time. If you attempt to add a p-Chip that already exists in the system, you will receive a warning from the application.
  • Your license must belong to an active Group in order to create records.
  • It is recommended to always specify a Product when creating a record.
  • A system generated noise will sound when a scan has been recorded.
  • If you are scanning multiple items, the list of records will be sorted from newest to oldest creation date/time. The last record scanned will be the first record in the list.

Edit Records

When a p-Chip record is created the External ID field is blank by default. This field can be updated by the user one time. Once this field is populated, it cannot be changed.

Instructions

  1. Open the Record Details window.
  2. Select the Edit button
  3. Enter the External ID in the External ID field
  4. Select Save

Add Products

A p-Chip record has a single product assigned to it at any given time. However, for multi-step processes, a single p-Chip can have multiple products assigned to it over its lifecycle. You can Add Products to an existing p-Chip record at anytime.

Instructions

  1. Connect your p-Chip wand to your device
  2. Select the Add Product button from the navigation menu
  3. If you belong to more then 1 group select the Group that should be the owner of this item, otherwise your default group will automatically be entered and not editable.
  4. Select the Product that should be added to the items you would like to scan from the dropdown menu.
  5. Select the Scanning Mode, multi-scan(recommended).
  6. Press start
  7. Press the tip of the wand against the p-Chip
  8. Stop scanner when all p-Chips have been scanned

Important!

  • When adding a product, the new product cannot be the same as the current product associated with the p-Chip you are scanning. If the products are the same, you will receive a notification and the action will be canceled.
  • Your license must belong to an active Group in order to add products.
  • After adding a Product, the following information will be visible on the Tag Details page:
    • Product Code
    • Product Name
    • Product Description
    • Product Status
    • Product Age
    • Production Date (UTC)
    • Production Location

Add Tests

Tests are supplmentary information that you can add to p-Chip Records. Tests are specified by your client administrator for your organization.

Instructions

  1. Connect your p-Chip wand to your device.
  2. Select the Add Test button from the navigation menu.
  3. Select the Scanning Mode, multi-scan(recommended).
  4. Press start.
  5. Press the tip of the wand against the p-Chip.
  6. Stop scanner when all p-Chips have been scanned.
  7. Select a row from the list of scans performed to open the Test Form.
  8. Select a Test from the dropdown list.
  9. Record your input in the Test Results/Comments section of the form.
  10. Press Save.

Important!

  • p-Chip records must exist in your organization in order to test them. If a record does not exist, you will be prompted to first Add the record before performing a test.

Search Records

p-Chips that have been applied to products and added in your can be searched using the p-Chip wand.

Instructions

  1. Connect your p-Chip wand to your device.
  2. Select the Search button from the navigation menu.
  3. Select the Scanning Mode, multi-scan(recommended).
  4. Press start.
  5. Press the tip of the wand against the p-Chip.
  6. Stop scanner when all p-Chips have been scanned.
  7. Select a row from the list of records you have searched to view the record details.

Important!

  • p-Chip records must exist in your organization in order to test them. If a record does not exist, you will be prompted to first Add the record before performing a test.

Report Issues

You can report issues with p-Chip hardware and software to your client administrators using the Error Report function.

Instructions

  1. Select the Error Report button from the navigation menu.
  2. Select the Type of Issue you are experiencing.
  3. Describe the issue you are experiencing in the Comments section.
  4. Press Save
